Узбекистана

У бизнесов нет ERP. Нет облачных баз данных. Нет структурированных данных.

Что есть на самом деле:

💾

Пиратская 1С

На компьютере бухгалтера. Без API. Без обновлений. Локальная база данных.

📊

Excel файлы

Разбросаны по рабочим столам, USB-флешкам и папкам "Новая папка (3)"

📄

Бумага и устные инструкции

Ключевые процессы передаются из уст в уста. Документооборот — бумажный.

💬

WhatsApp группы

Основной канал деловой коммуникации. Заявки, отчёты, согласования — всё тут.

Даже крупные компании как Musaffo (ведущий молочный завод страны) — работают вслепую. CFO: "Мы не знаем своих цифр"

Контекст

Почему облачный SaaS
не работает здесь

  • ✕ Нельзя «подключиться к PostgreSQL» — его просто нет
  • ✕ Нельзя загрузить данные в облако — данные живут на локальной машине рядом с пиратской 1С
  • ✕ 1С пиратская — нет API, нет облачной синхронизации, нет интеграций
  • ✕ ERP стоит $10,000+ на внедрение — нет местных специалистов для настройки
  • ! Сотрудники сопротивляются переменам — урок Binaar: продукт работал, adoption = 0
  • ! Бизнес «работает» на 1С + Excel + телефонные звонки. И менять это никто не готов

Весь SaaS-мир строит для компаний с облачной инфраструктурой. В Узбекистане этой инфраструктуры нет. Нужен другой подход.

Rendesk не заменяет рабочий процесс. Он садится поверх того, что уже есть — 1С, Excel, WhatsApp — и делает это умнее.

Не нужно менять 1С. Не нужно учить сотрудников новой системе. Просто задайте вопрос — получите ответ.

Возможности

Что умеет Rendesk

01

Читает данные из 1С

Подключается к локальной 1С через COM. Вытаскивает любые данные без SQL и без знания схемы базы.

"Покажи остатки на складе на сегодня"
02

Создаёт документы динамически

Генерирует Excel, PDF, Word — любой формат. Отчёты, накладные, акты сверки — по запросу на русском языке.

"Сделай отчёт по складу за март" → Excel на рабочем столе
03

Записывает данные в 1С

Может заносить данные обратно в 1С — накладные, проводки, документы. Без ручного ввода.

"Добавь эти 47 накладных из Excel в 1С" → готово за минуту
04

Отвечает на вопросы о бизнесе

Запрашивает данные из 1С, считает, анализирует — и отвечает на русском языке.

"Сколько потратили на сырьё в феврале?" → ответ за 30 секунд
05

Адаптируется под каждый бизнес

Нет предустановленных процессов. AI сам разбирается в структуре данных и подстраивается.

Молочный завод, стройка, торговля — один продукт, разные задачи
Архитектура

Почему это должно
работать локально

Ценность — в локальном выполнении на машине, где живут данные

💻

1С работает только локально

Пиратская 1С не имеет облачного API. Единственный способ получить данные — через COM-объекты на том же компьютере, где установлена 1С.

🔒

Данные не покидают офис

Нет доверия к облаку. Вопросы с пиратским ПО. Клиенты не готовы отправлять бухгалтерские данные на чужие серверы.

📁

Доступ к файловой системе

Excel файлы, накладные, акты — всё лежит на локальном диске. Rendesk видит те же папки, что и бухгалтер.

⚡

Мгновенный доступ к данным

Нет задержек сети. Нет зависимости от интернета. Desktop-приложение работает рядом с 1С на том же компьютере.

Rendesk Desktop устанавливается на компьютер бухгалтера.
Данные остаются в офисе. Всегда.
